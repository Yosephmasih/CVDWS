Colour Vision Deficiency Simulation Website

Colour Vision Simulation Website is a website that will enable users, and developers to simulate colours as the colour-blind would perceive. This project simulates PDFs, Images, and URLs by using the Mozilla Firefox extension. Feel free to use the website, test the features, and send recommendations, and even compliments! I hope this simple project will serve your needs.

-- Project --

Common Colour Defects:


  • Normal Vision - The person has no defects

  • Protanopia - Inability to perceive Red light

  • Deuteranopia - Inability to perceive green light.

  • Tritanopia - Inability to perceive blue light.

  • Achromatopsia - Reduced sensitivity to all colors.
